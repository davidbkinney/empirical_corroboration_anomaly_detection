# Data and Code Repository for "Empirical Corroboration of Anomaly-Detection Strategies in Astrobiology"

This repository contains code and data needed to replicate the analysis contained in the paper "Empirical Corroboration of Anomaly-Detection Strategies in Astrobiology: Evidence from Amino Acid Abundance Distributions and Mass Spectroscopy Datasets" by Kinney, Allmendinger-Tilson, and Davila.
